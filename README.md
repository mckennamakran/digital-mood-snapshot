# 🌱 Digital Mood Snapshot (Java)

## 📖 Overview
A simple **beginner-friendly Java program** that captures a user's emotional state and displays a personalized “mood snapshot”.

This project focuses on **core Java fundamentals** such as variables, constants, user input, and basic validation, while also promoting **mental health awareness**.

---

## 🎯 Features
- 🧑 Capture user details (name & mood)
- ⚡ Record energy level on a scale of 1–10
- 📋 Display a clean “Mood Snapshot” summary
- 🛑 Basic input validation for energy levels
- 💡 Includes a predefined mood list for guidance

---

## 🧠 Concepts Used
- Variables (`String`, `int`)
- Constants (`final`)
- User Input (`Scanner`)
- Arrays (`String[]`)
- Conditional Statements (`if`, `else if`)
- Basic Output Formatting

---

## 🛠️ How It Works
1. The program asks the user to enter:
   - Their **name**
   - Their **current mood**
   - Their **energy level (1–10)**

2. It provides a sample list of moods:
   - Happy, Sad, Calm, Angry

3. It validates the energy level:
   - Must be between **0 and 10**

4. Finally, it displays a formatted mood report.
